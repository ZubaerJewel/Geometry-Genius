Java Script question and answer

1.What is DOM and what is the purpose of DOM?

DOM is known as Document Object Model. which is a programming interface for web documents.It represents the page so that programs can change the document structure, style,element and content.

2.How will you select HTML elements using DOM? Name the DOM methods.

I have to Select HTML DOM elements by the name attribute, using the JS getElementsByTagName(), getElementById() and getElementsByClassName() ,querySelector (), querySelectorAll() method.

3.What is event bubble?

It is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy.

4.What is a callback function and why will you use it?

A callback's purpose is to execute code in response to an event. These events can be like mouse clicks; with callback, we can add text at the end of each statement like execute this code every time the user clicks a key on the keyboard.
